# Сравнительный анализ сервисов геокодирования

## Общая информация

**Цель исследования:** выбрать наиболее подходящий сервис для выполнения прямого и обратного геокодирования в скоринговой платформе.

В ходе работы были исследованы API следующих сервисов: Яндекс.Геокодер, DaData, GigaChat.

Исследование выполнялось путем изучения документации сервисов, автоматизированного выполнения запросов при помощи Python-скрипта, анализа полученных результатов.

Для тестирования использовались:

* 100 адресов для прямого геокодирования;
* 100 координат для обратного геокодирования.

Всего было выполнено **600 запросов** (200 на каждый сервис).

## Критерии оценки

| Критерий                                                   | Описание                                                                                  |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| Поддержка прямого геокодирования     | преобразование адреса в координаты                                 |
| Поддержка обратного геокодирования | преобразование координат в адрес                                     |
| Скорость ответа                                      | среднее время выполнения запроса                                     |
| Точность результата                              | наличие информации о качестве найденного результата |
| Простота интеграции                              | сложность работы с API                                                            |
| Назначение сервиса                                | соответствует ли сервис задаче геокодирования            |

## Анализ сервисов

### Яндекс.Геокодер

#### Преимущества

* специализированный сервис геокодирования;
* поддерживает прямое и обратное геокодирование;
* высокая скорость работы;
* предоставляет уровень точности;
* простой REST API.

#### Недостатки

* отсутствует очистка адресов;
* менее удобен при работе с неструктурированными адресами.


### DaData

#### Преимущества

* очистка и нормализация адресов;
* получение координат;
* поддержка обратного геокодирования;
* наличие показателя качества определения координат.

#### Недостатки

* более сложная интеграция (используются разные API для различных сценариев);
* среднее время ответа выше, чем у Яндекс.Геокодера.


### GigaChat

#### Преимущества

* способен определять координаты и адреса по текстовому запросу;
* не требует специализированного API геокодирования.

#### Недостатки

* не является специализированным сервисом геокодирования;
* отсутствуют гарантии точности результатов;
* отсутствуют показатели качества геокодирования;
* ответы формируются языковой моделью и могут изменяться;
* требуется получение OAuth-токена.


## Вывод

По результатам проведенного анализа наиболее подходящим сервисом для использования в скоринговой платформе является **Яндекс.Геокодер**.

Причины выбора:

* специализированный API для геокодирования;
* самая высокая скорость ответа среди протестированных сервисов;
* поддержка прямого и обратного геокодирования;
* наличие информации о точности найденного объекта;
* простая интеграция.

Сервис **DaData** целесообразно использовать как дополнительный источник для нормализации и очистки пользовательских адресов перед отправкой в геокодер.

**GigaChat** не рекомендуется использовать в качестве основного механизма геокодирования, поскольку он является языковой моделью, а не специализированным геоинформационным сервисом.
